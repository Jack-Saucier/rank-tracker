# Rank Tracker — Step 1 Starter

This is the smallest possible working version of the app: one page that says hello.
The goal right now isn't features — it's getting *something* live on the internet.

## Run it locally
```
npm install
npm run dev
```
Then open http://localhost:3000

## Deploy it (see the walkthrough in chat for full steps)
1. Push this folder to a GitHub repo
2. Import that repo on vercel.com
3. Click Deploy
